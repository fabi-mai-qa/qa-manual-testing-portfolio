This repository contains a complete manual QA framework created to demonstrate
my experience in software quality assurance, including test strategy, test plans,
test cases, bug reports and test summary reports.

## Scope
- Functional testing
- API testing
- Test documentation

## Tools and Skills
- Manual Testing
- API Testing (Postman)
- SQL (Oracle, SQL Server)
- ISTQB CTFL 4.0

## Author
Software QA Analyst with experience in manual functional tests in desktop, web, mobile platforms and API tests using Postman.
