# TC-004.01 - Create Order with single Seller, single Product and no Shipping Fee

## Description  
Validate successful order creation with a single seller and a single product, without applying a shipping fee.

## Preconditions  
- User is authenticated with a valid API token.
- Seller exists with valid shipping configurations.
- Product exists and is linked to the seller.

## Endpoint
/orders  

## Payload
```json
{
  "userId": 123,
  "orderItems": [
    {
      "productId": 10,
      "quantity": 2,
      "unitPrice": 50.00,
      "sellerId": 145
    }
  ],
  "shippingBySeller": [
    {
      "sellerId": 145,
      "hasShippingFee": false,
      "shippingType": "STANDARD",
      "estimatedDeliveryDays": 7
    }
  ]
}
```

**Expected Result:**  
- Order is created successfully.
- HTTP Status Code: 201 Created.
- No shipping fee is applied.
- Total price is calculated as:
	```text
	 2 × 50.00 = 100.00
	```
- A unique `orderId` is returned in the response.

**Business Rules Covered:**
- Shipping fee applied only when `hasShippingFee = true`.
- Total price calculation without shipping fee value.
- Total order price is calculated by the system when not provided in the request.
 
**Status:**  
- Not Executed / Passed / Failed
